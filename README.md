# Weather App

### **Overview - About this project**
This is a simple Weather App using Clean Architecture (MVI, Use Cases, Hilt)  and also Jetpack Compose to show a modern **UI/UX**

Used API:
- OPEN METEO API
https://open-meteo.com/
